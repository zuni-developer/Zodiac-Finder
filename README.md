# ♈ Zodiac Finder

A simple yet visually polished **Zodiac Finder web app** built with **HTML, CSS, and JavaScript**.  
Users enter their name and date of birth to instantly reveal their zodiac sign, presented with a clean cosmic-themed UI.

This project focuses on **good UI, smooth UX, and clarity**, rather than heavy logic or frameworks.

---

## Features

- Takes user **name & date of birth**
- Calculates the **correct zodiac sign**
- Shows results on a **separate result page**
- Uses **localStorage** to pass data between pages
- Each page has its **own background and vibe**

## How to Run

1. Clone the repository  
   ```bash
   git clone https://github.com/zuni-developer/Zodiac-Finder
    ```

2. Open the folder
3. Run using **Live Server** or open `index.html` directly
